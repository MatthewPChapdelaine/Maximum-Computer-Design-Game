# Changelog

All notable changes to the Maximum PC Builder game will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-13

### Added
- Initial release of Maximum PC Builder game
- Progressive research system with tech tree
- 40+ components across 8 categories (CPU, RAM, Storage, GPU, Infrastructure, Network, Power, Cooling)
- Real-time resource management (money, research points, power budget)
- Performance scoring system
- Save/Load game functionality
- Dark-themed GUI optimized for extended play
- Component prerequisite system
- Progress bars for research tracking
- Category filtering for component browsing
- Ultimate build detection and victory condition
- Passive income and research point generation
- Power budget management
- .deb package for easy installation on Debian-based systems
- Desktop entry with application launcher integration
- Comprehensive documentation

### Components Include
- CPUs: Budget to AMD EPYC 9754 and Intel Xeon Platinum 8592+
- RAM: 16GB DDR4 to 24TB DDR5-6400 configurations
- Storage: 500GB SATA SSD to 1PB NVMe RAID arrays
- GPUs: Budget GPUs to 8x NVIDIA H200 and 4x B200 configurations
- Complete infrastructure components
- Networking from 1Gb to 100Gb
- Power supplies from 750W to 24kW
- Cooling solutions from basic to server-grade

### Features
- Strategic planning required for efficient progression
- Real-world inspired component specifications
- Educational value about PC hardware relationships
- Engaging progression mechanics
- City-building style resource management
